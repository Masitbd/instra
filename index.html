What is the difference between localStorage and sessionStorage?

Most mordern browsers support local storage and session storage. Bothe use for same purpose like storage data in the browser. Each browser have differnt specification to store data but most of poputar browser have capacity 5-10 mb.

The main difference between local storage and session storage is that local storage has no expiration but session session storage data is gone, then you close the browser tab. Both stores are accessible via javascript DOM. Create key/value pair localStorage.setItem provides key and value. We can use localStorage or sessionStorage based of our needs.
